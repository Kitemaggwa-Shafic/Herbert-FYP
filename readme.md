# Herbet FYP 
Herbert FYP Students Library Management System (OSLMS)

## Description
This project is a student final year project that allows students to have access to studnt material online. The project is built using the latest web technologies, such as HTML5, CSS3, JavaScript, and jQuery. The project has a modern and clean design, with easy navigation and a user-friendly interface. The project is responsive and works on all devices.
## Installation
To run this project, you need to have a web server, such as Apache installed on your computer. You also need to have PHP and MySQL installed. Once you have these requirements installed, follow these steps:
1.Clone the repository to your local machine.
2.Import the database using the provided SQL db file.
3.Update the database configuration in the config.php file.
4.Launch the project on your web server.
## Usage
The project has the following features:
1. Student registration and login where students register first and login.
2. You will login with either details of student or with details of Lecturer depeding on your account details registered in the system.
3. Browse available reading material  by categories.
4. Search for available rading material by name, course or description  etc.
5. Lecturers can add reading material for students to use
6. Lectures can approve the students sent in material before this material is published for other students to use, can go through it before approving
7. Students can download the available aprroved material for future reference even when not online.
8. The student can only access material uploaded by the lecturer in the system and can download, upload, edit their work but only if they are in their accounts.
